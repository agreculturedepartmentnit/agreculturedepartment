About this Website

This is a ready-to-use academic website template for a “Soil Map of Tamil Nadu” project. It contains:

  • Interactive map (Leaflet) with district boundaries
  • Soil type pages with organized sections
  • District list page + district detail pages
  • State soil proportion chart from a JSON dataset

How to make it complete (recommended)

  1. Replace the sample tn_districts.geojson with a real Tamil Nadu districts GeoJSON.
  2. Add an official soil layer: either download soil polygons (vector) or connect a WMS and export polygons.
  3. Compute district-wise soil % in QGIS and export as JSON (already supported by this template).
  4. Add references/download links (Soil Atlas PDFs, survey reports) in this page.
